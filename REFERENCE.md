## When to update this file

Update `REFERENCE.md` when you:

- Add a new file in `prototype/src/`, `prototype/server/`, `docs/`, `scripts/`, or `.claude/`.
- Move or rename a tracked file.
- Delete a tracked file.
- Convert an "Empty" directory to one with content (replace the placeholder line with real entries).

Keep entries to **one line**. If a file needs more context, link to it from `docs/PLAN.md` or `docs/DECISIONS.md`, not here.
